关键点在这个部分:

当terminal使用echo来输出内容到终端时，有自己的语法用于在终端做渲染

例如我写了这样一个函数

$(function() {
  $('#myterm').terminal({
    hello: function(name) {
      this.echo('[[b;red;background]Hello ]' + name);
    },
    add: function(a, b) {
      this.echo(Number(a) + Number(b));
    }
  }, {
    greetings: '欢迎!',
    prompt: '> '
  });
});

接着我们输入hello 1就会看到hello首字母大写了并且颜色变红
